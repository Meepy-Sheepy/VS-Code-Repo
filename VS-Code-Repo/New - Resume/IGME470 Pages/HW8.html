Keeping Track Of Time

Today, I'm going to be working on Chapter 8: Digital Hourglass. You know when you have a regular hourglass that has sand inside of it and you flip it around to restart the timer? I'm going to be doing that but digitally! With LEDs!

Instead of using sand to track the time, I'm going to use LEDs. As the time increases, the amount of LEDs that are on will increase. For example, if 20 seconds has passed and I wanted an LED to light up every 10 seconds, then 2 LEDs would be on. Enough chatter, let's get right into it!

A Light Show

The circuitry is actually kind of basic. It's very similar to Chapter 1, where I just used 1 LED to turn it on. I put in 6 LEDs, each of them representing 10 seconds. These are our outputs. The new item for this chapter is the tilt sensor. The tilt sensor is like a regular switch with an on/off sensor, but instead there is a tiny ball inside it that detects orientation. So when we tilt it, it knows that it has been moved. The tilt sensor is our input. I'm going to be using the data from the tilt sensor to be able to reset the LEDs and timer.
